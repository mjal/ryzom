ryzom.ddp module¶

Functions to communicate DDP messages to the channel layer.

ryzom.ddp.send_change(sub, model, tmpl, _id)[source]¶

Send change message. Function used to send a DDP message to a specific client via the channel layer. Uses the template class associated with a publication to create a new instance of a component attached to a model that was updated Essentially called by post_save and post_delete signal handlers

Parameters:
  • sub (Subscriptions) – The Subscription holding the connection information
  • model (Publishable) – The class of the model to change
  • tmpl (Component) – The component subclass that templates the model instance
  • _id (int) – The id of the model to change
ryzom.ddp.send_insert(sub, model, tmpl, _id)[source]¶

Send insert message. Function used to send a DDP message to a specific client via the channel layer. Uses the template class associated with a publication to create a new instance of a component attached to a model that was inserted, updated or removed Essentially called by post_save and post_delete signal handlers

Parameters:
  • sub (Subscriptions) – The Subscription holding the connection information
  • model (Publishable) – The class of the model to insert
  • tmpl (Component) – The component subclass that templates the model instance
  • _id (int) – The id of the model to insert
ryzom.ddp.send_remove(sub, model, tmpl, _id)[source]¶

Send remove message. Function used to send a DDP message to a specific client via the channel layer. Uses the template class associated with a publication to create a new instance of a component attached to a model that was removed, in order to get the computed _id and send the computed _id to the client. Essentially called by post_save and post_delete signal handlers

Parameters:
  • sub (Subscriptions) – The Subscription holding the connection information
  • model (Publishable) – The class of the model to remove
  • tmpl (Component) – The component subclass that templates the model instance
  • _id (int) – The id of the model to remove
