ryzom.reactive module¶

Defines the ReactiveComponent class to be inherited to create reactive content

class ryzom.reactive.ReactiveComponent(name, view, tag='div', content=[], attr={}, events={}, parent='body', _id=None)[source]¶

Bases: ryzom.components.Component

Subclass of component with reactive content. It takes a unique name and a ryzom.views.View in addition to common component parameters.

Parameters:
  • name (str) – The (unique) name of the instance
  • view (View) – The rizom View this instance is attached to
setcontent(content)[source]¶

This method should be called only by the associated view to update the component contents, then it sends the new content to the client associated with the view instance
